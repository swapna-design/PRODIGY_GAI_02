# PRODIGY_GAI_02
Generated images from text prompts using a pre-trained Stable Diffusion model and explored the fundamentals of AI-based image generation.

#Overview

In this project, I used a pre-trained Stable Diffusion model to generate images from text prompts. The model converts natural language descriptions into realistic and creative images using Generative AI techniques.

Python
Google Colab
Stable Diffusion
Diffusers Library
#Outcome

Successfully generated AI-created images from text prompts using a pre-trained image generation model.
